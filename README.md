## 🛠 Run Locally

### Prerequisites

* Node.js installed on your system

### Installation

1. Clone the repository

```
git clone https://github.com/aryankahar31/public-wifi-security-awareness-portal.git
```

2. Navigate to the project directory

```
cd public-wifi-security-awareness-portal
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```
